Benutzeranleitung - Gruppenverwaltung & Nachrichtenversand

Über diese Anwendung

Diese Webanwendung ermöglicht die Verwaltung von Gruppen und den gezielten Versand von Nachrichten. Sie wurde speziell für die DRV-Bund entwickelt.

Hauptfunktionen

Nachrichten senden

Versenden Sie gezielt Nachrichten an ausgewählte Gruppen oder alle Gruppen gleichzeitig.

Gruppen hinzufügen

Erstellen Sie neue Gruppen für eine bessere Organisation Ihrer Kontakte.

Gruppen löschen

Entfernen Sie nicht mehr benötigte Gruppen einzeln oder mehrere gleichzeitig.

Intelligente Suche

Nutzen Sie die Suchfunktion, um schnell die gewünschten Gruppen zu finden.

Schritt-für-Schritt Anleitung

1. Nachrichten senden

  1. Klicken Sie im Menü auf "Nachrichten"
  2. Wählen Sie den gewünschten Jahrgang aus
  3. Wählen Sie "Studenten" oder "Auszubildende"
  4. Selektieren Sie die Zielgruppen (oder aktivieren Sie "An alle senden")
  5. Verfassen Sie Ihre Nachricht
  6. Klicken Sie auf "Senden"
  7. Die Nachricht wird als JSON-Datei heruntergeladen

2️. Neue Gruppe hinzufügen

  1. Klicken Sie im Menü auf "hinzufügen"
  2. Geben Sie den Gruppennamen ein
  3. Drücken Sie Enter oder klicken Sie "Hinzufügen"
  4. Die Gruppe wird automatisch zur Datenbank hinzugefügt

3️. Gruppen löschen

  1. Klicken Sie im Menü auf "Löschen"
  2. Wählen Sie eine oder mehrere Gruppen aus der Liste
  3. Drücken Sie Enter oder klicken Sie "Löschen"
  4. Die ausgewählten Gruppen werden permanent entfernt

Tipps & Tricks

Schnelle Suche: Nutzen Sie die Suchfunktion in den Dropdown-Menüs, um große Listen schnell zu durchsuchen.
Tastenkürzel: Drücken Sie Enter auf jeder Seite, um die Hauptaktion auszuführen.
