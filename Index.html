Benutzeranleitung - Gruppenverwaltung & Nachrichtenversand

Über diese Anwendung

Diese Webanwendung ermöglicht die Verwaltung von Gruppen und den gezielten Versand von Nachrichten. Sie wurde speziell für die DRV-Bund entwickelt.

Schritt-für-Schritt Anleitung

Nachrichten senden

  1. Klicken Sie im Menü auf "Nachrichten"
  2. Wählen Sie den gewünschten Jahrgang aus
  3. Wählen Sie "Studenten" oder "Auszubildende"
  4. Wählen Sie die Zielgruppen (oder aktivieren Sie "An alle senden")
  5. Verfassen Sie Ihre Nachricht
  6. Klicken Sie auf "Senden"
  7. Die Nachricht wurde versendet

Neue Gruppe hinzufügen

  1. Klicken Sie im Menü auf "hinzufügen"
  2. Geben Sie den Gruppennamen ein
  3. Drücken Sie Enter oder klicken Sie "Hinzufügen"
  4. Die Gruppe wird automatisch zur Datenbank hinzugefügt

Gruppen löschen

  1. Klicken Sie im Menü auf "Löschen"
  2. Wählen Sie eine oder mehrere Gruppen aus der Liste
  3. Drücken Sie Enter oder klicken Sie "Löschen"
  4. Danach drücken Sie dann Bestätigen
  5. Die ausgewählten Gruppen werden permanent entfernt
